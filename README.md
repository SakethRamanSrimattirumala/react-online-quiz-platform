# React Online Quiz Platform

## Project Overview

An interactive single-page quiz application built using ReactJS with dynamic routing and modular component architecture. The application enables seamless navigation between quiz stages and provides an engaging user experience through interactive UI design.

## Features

- Dynamic routing using **React Router** (**Home, Quiz, Results**)
- Interactive quiz flow with **state-based answer selection**
- Smooth navigation across quiz screens
- Modular and reusable React component architecture
- Responsive and clean UI styling using CSS

## Tech Stack

- HTML
- CSS
- ReactJS
- React Router

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/react-online-quiz-platform.git
```

Navigate to project folder:

```bash
cd react-online-quiz-platform
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
